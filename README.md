# langchain-groq-cli-agent
A CLI-based conversational AI agent built with LangChain and Groq LLM, featuring real-time web search using the Google Serper API.

LangChain Groq CLI Agent is a command-line based conversational AI agent built using LangChain and Groq LLM.
The agent can interact with users through the terminal and answer questions, including performing real-time web searches using the Google Serper API.

This project demonstrates how to integrate a large language model with external tools in a simple CLI environment, making it beginner-friendly and easy to understand.
